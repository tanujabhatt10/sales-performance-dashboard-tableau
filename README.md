📌 Project Overview

This project is an interactive Sales Performance Dashboard created using Tableau. The dashboard provides insights into sales performance through interactive visualizations, filters, and dashboard actions.

🎯 Objectives

- Analyze sales performance by category.
- Track monthly revenue trends.
- Identify the top-performing products.
- Understand category-wise revenue contribution.
- Compare product quantity across categories.

📈 Dashboard Visualizations

- Revenue by Category (Bar Chart)
- Monthly Revenue Trend (Line Chart)
- Top 5 Products by Revenue
- Revenue Share by Category (Pie Chart)
- Quantity by Category (Bar Chart)

✨ Features

- Interactive Dashboard
- Category Filter
- Dashboard Actions (Use as Filter)
- Sorting (Ascending/Descending)
- Top N Filter
- Percent of Total
- Professional Dashboard Layout

🛠️ Tools Used

- Tableau Public
- Microsoft Excel

📊 Key Insights

- Identified the highest revenue-generating category.
- Analyzed monthly revenue trends.
- Identified the top 5 revenue-generating products.
- Compared revenue contribution by category.
- Compared quantity sold across categories.
